Atividade 1: Criando um banco de dados

No terceiro bimestre do terceiro ano, estudamos o tema "banco de dados". Este tema se destacou muito, e no meu ponto de vista, foi o tema mais fácil estudado dentro do terceiro ano inteiro.
Começamos utilizando a plataforma "Access", onde ela tráz familiaridade com o Excel. No entanto, a utilização de banco de dados, foi bem simples, apesar de não ser uma plataforma comumente usada, ela traz limitações e simplicidade dos recurso. Podemos chamar o Access de uma introdução ao banco de dados.
